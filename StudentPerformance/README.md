# 📊 About Dataset

## 📖 Description

The **Student Performance Dataset** is designed to examine the factors that influence academic student performance. The dataset contains **10,000 student records**, with each record including information about several predictor variables and a performance index.

This dataset aims to help researchers, students, and data analysts explore the relationship between different academic and lifestyle factors and overall student performance.

La base de datos esta aquí {https://www.kaggle.com/datasets/nikhil7280/student-performance-multiple-linear-regression}

---

# 📌 Variables

### 📚 Hours Studied

The total number of hours spent studying by each student.

### 📝 Previous Scores

The scores obtained by students in previous tests.

### 🎭 Extracurricular Activities

Indicates whether the student participates in extracurricular activities (**Yes** or **No**).

### 😴 Sleep Hours

The average number of hours of sleep the student gets per day.

### 📄 Sample Question Papers Practiced

The number of sample question papers practiced by the student.

---

# 🎯 Target Variable

## Performance Index

The **Performance Index** is a measure of a student's overall academic performance.

### Characteristics

* Rounded to the nearest integer.
* Ranges from **10 to 100**.
* Higher values indicate better academic performance.

---

# 🔍 Purpose of the Dataset

The dataset is intended to provide insights into how different factors may affect student performance.

Researchers and data analysts can use it to study the impact of:

* 📚 Study time
* 📝 Previous academic achievement
* 🎭 Participation in extracurricular activities
* 😴 Sleep habits
* 📄 Practice through sample question papers

on students' overall academic outcomes.
